# Prompt for AI

Kho lưu trữ **các prompt (lời nhắc) dùng cho AI** — Claude, ChatGPT, Gemini, v.v.

Đây là nơi tập hợp lại tất cả các prompt hữu ích cho nhiều mục đích khác nhau: học tập, lập trình, viết lách, phân tích, công việc hằng ngày... Mỗi prompt là một file Markdown độc lập, có thể copy-paste trực tiếp để sử dụng.

## Danh sách prompt

| File | Mục đích |
|------|----------|
| [deep-understanding-tutor.md](deep-understanding-tutor.md) | Biến AI thành gia sư dạy hiểu sâu — đảm bảo người học thực sự nắm vững thay vì học vẹt. |
| [paraphraser.md](paraphraser.md) | Biến AI thành chuyên gia cải biên (paraphrase) văn bản chuyên nghiệp với nhiều phong cách khác nhau. |

> Danh sách sẽ được bổ sung dần khi có thêm prompt mới.

## Cách sử dụng

1. Mở file prompt mong muốn trong repo.
2. Copy toàn bộ nội dung.
3. Dán vào AI (Claude, ChatGPT...) làm **system prompt** hoặc tin nhắn đầu tiên.
4. Bắt đầu tương tác.

## Đóng góp / Thêm prompt mới

1. Tạo một file `.md` với tên mô tả rõ mục đích (dùng `kebab-case`, ví dụ `code-reviewer.md`, `email-writer.md`).
2. Viết nội dung prompt rõ ràng, súc tích.
3. Thêm một dòng vào bảng **Danh sách prompt** ở trên.

## Quy ước

- Mỗi file = một prompt.
- Tên file mô tả đúng chức năng, viết theo `kebab-case`.
- Ưu tiên tiếng Việt hoặc tiếng Anh tùy ngữ cảnh sử dụng của prompt.
